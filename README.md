# thunes

### 功能
1. 用户登录（分配JWT作为身份验证）
2. 用户的个人信息及账户查询
3. 用户转账

### 功能验证测试
- 文档地址：http://localhost:8099/v2/docs/index.html
- 服务启动：在项目根目录执行 docker-compose up

> 1. 服务默认运行于8099端口，可在thuens.toml中配置（若修改则需要更新对应的接口文档服务器地址）
> 2. msyql服务依赖3306、33060端口，请确保端口未被占用
